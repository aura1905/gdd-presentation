필드 전투 시스템 개요

Core Loop

핵심 루프: 필드 진입 → 적 조우/토벌 → 헥스 점령 → 물자 생산 → 가문 성장 → 다음 레기온 진출

필드 이동 (자유) ├─ 조우전: 이동 중 확률/조건으로 만남 (수동적) ├─ 공격전: 플레이어가 직접 개시 (능동적) └─ 방어전: NPC 세력이 쳐들어옴 (반응적) 점령 구조: 이동만 (통과) → 물자 생산 없음 조우전 승리 → EXP, 소량 보상 점령 토벌 승리 → 헥스 점령, 물자 생산 개시 구조물 공성 승리 → 거점/관문/도시 확보

구조물 라이프 (성장 시스템)

점령 후 물자 투자로 구조물을 강화하면 방어병력이 성장.

캐릭터 부상 시스템

가챠 획득 캐릭터는 절대 영구 사망하지 않음.

부상은 일시적 디버프이며, 시간 또는 자원으로 치료 가능.

부상 등급발생 조건스탯 감소출전치료 방법

중상패배 시 HP 0-30%가능 (약화)거점에서 약재+곡물 (3시간)

삼전 참조

핵심 원칙: 피로도 = ATK 직결 감소 (DEF 유지). 거점 관리와 퇴각 판단이 전략의 핵심.

모든 이동은 실시간 소요. 이동 명령은 경로 지정 후 자동 진행.

퇴각 시스템

안전 복귀

퇴각 = 전투 없이 출발 거점으로 안전 복귀. 왔던 경로를 역순 이동.

이동 시간은 동일하게 소요되지만, 피로도 추가 누적 없음.

항목규칙
발동 조건전투 밖에서 언제든 가능
목적지마지막으로 휴식한 거점 / 도시 / 벙커
이동 시간정상 이동과 동일 (칸당 2~4분)
피로도 변화추가 없음 현재 수치 유지
전투 발생없음 안전 복귀
중단불가 퇴각 명령은 취소 불가 — 거점 도착까지 자동 진행

피로도 회복

구조물 전용

피로도는 구조물(거점/도시/벙커)에서만 회복 가능.

음식·야영·과금 등으로 피로도를 회복할 수 없음. 거점 확보와 벙커 건설이 진격의 핵심 전략.

거점 / 도시

100%

즉시 전회복. 가장 안전하고 확실한 방법.

벙커

50%

즉시 회복. 자원 소모하여 건설 필요 (목재 500 + 석재 300 + 식량 200).

§13 연동

벙커는 피로 회복(50%) + 주변 3칸 물자 임시 저장. 자원 밀집 지역에 벙커를 배치하면 수거 효율 극대화.

단, 벙커도 슬롯 1 소모 + 곡물 유지비(10/h) → 무한 건설 불가.
