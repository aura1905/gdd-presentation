5. ⭐ 원정 시스템 (신규) — 점령 → 수집

핵심: 거점에서 출진 → 웨이포인트 시퀀스 따라 이동/전투 → 전리품 모음 → 거점 복귀 → 창고 저장. 영구 점령 없음.

5-1. 흐름

[거점] 출진 준비 ↓ [웨이포인트 1] 전투 → 승리 → 전리품 (캐릭이 들고 있음) ↓ [웨이포인트 2] 전투 → 승리 → 전리품 ↓ ... (가문 성장으로 해금된 N개) ↓ [거점 복귀] → 전리품 창고 저장 (안전)

5-2. 패배 처리 — 50% 드롭 + 1시간 잔존

  1. 패배 시 파티 강제 후퇴, 거점으로 복귀
  2. 그 시점까지 모은 전리품 50%만 그 자리에 잔존 (나머지 50%는 즉시 소실)
  3. 잔존 전리품은 1시간 동안만 유지 → 다른 파티로 재방문 회수 가능
  4. 1시간 경과 → 영구 소실

5-3. 가문 성장 연동

성장 항목효과
정찰 능력이동 가능 거리 +
원정 경험웨이포인트 수 +
보급 능력전리품 보유 한도(인벤 캡) +
후퇴 훈련패배 시 손실률 50% → 더 적게
회수 시간잔존 시간 1h → 더 길게

6-A. 🏰 알쿠에트 모레자 던전 — 5월 마감 도달 목표

GE 원작의 알쿠에트 모레자(Al Quelt Moreza) 던전을 5/15 마감의 최종 콘텐츠 도달점으로 한다. 신규 유저가 진입 후 이 던전을 클리어할 수 있어야 5월 구현 완료.

6-A-1. 컨셉

  • 일반 원정의 확장형 — 더 긴 웨이포인트 시퀀스 + 마지막 보스 웨이포인트
  • 리볼도외에서 출진 → 알쿠에트 모레자 도달 → 보스 처치 → 클리어 보상 + 챕터 진행
  • 가문 성장 / 캐릭 성장이 어느 정도 진행되어야 클리어 가능한 난이도

6-A-2. 데이터 (신규)
