게임 개요

CORE CONCEPT

장르: 가문 경영 전략 RPG (풋볼매니저 × 삼국지전략판 × HoMM × GE IP)

핵심 컨셉: GE 원작의 395명 캐릭터를 영입·육성·편성하여 신대륙에서 가문을 경영하고 세력을 확장하는 전략 게임.

배경: 레콩키스타 시대(1522~) — 3년 전쟁 이후 신대륙 개척 경쟁이 격화된 시기.

🎯 핵심 설계 철학 — "가문 경영 서사"

캐릭터 = 전투 도구가 아닌, 가문의 식구 영입하고 → 같이 살고 → 역할을 주고 → 밥을 먹이고 → 관계가 형성되고 → 함께 전쟁하고 → 때론 떠나보내고 → 가문의 역사가 쌓이는 것 GE 원작의 혁신적 MCC(다캐릭터 동시 조작) + 395명 영입 캐릭터 + 가문 시스템의 정신을 계승하되, 삼전/FM/HoMM의 체계적 시스템 적용.

🎮 핵심 설계 원칙

원칙 설명
전투보다 운영 전투는 결과를 보는 것. 운영(편성/배치/경영)이 핵심 재미.
캐릭터 = 가문의 식구 도구가 아닌 관계의 대상. 충성도·사기·관계가 성과에 영향.
레벨 깡패 없음 조합·상성·시너지가 승패 결정. 약해도 상성 좋으면 이긴다.
캐릭터는 항상 부족 공격 + 수비 + 토벌 + 생활 동시에 → 더 영입, 더 육성.
PvF (Player vs Faction) PvP 피로감 없는 NPC 세력 전쟁. 적의 적은 친구.
감정적 과금 "저 세력을 꺾으려면 이 캐릭터가 필요" → 전략적 필요에 의한 과금.

필드 탐색 시스템

거점-관할 지역 모델

핵심 컨셉: 거점 중심 영토 확장. 보유 거점에서 출발 → 관할 지역 타일 탐색 → 자원 수집/이벤트 → 새 거점 발견·편입.

HoMM 차별화: 단순 정복이 아닌, 거점 = 배럭 + 자원 + 관할 지역. FM식 경영과 결합.

📊 10턴 전개 — 3막 구조

턴 1~3: 개전기

"누가 먼저 치느냐"

SPD 높은 쪽이 선공. 버프/디버프 깔기. 미사일 조합은 여기서 끝내려 함.

턴 4~7: 중반

"구도가 정해진다"

3:3 → 3:2 또는 2:2로. 수적 우위가 전투를 갈라놓음. CC 제어 조합의 전성기.

턴 8~10: 마무리

"역전 또는 확인사살"

HP 위기 시 스킬 발동률 +10%. 지구전 조합의 진가 발휘.

💥 최종 데미지 공식 — 삼전 기반

주피증은 합연산, 받피감은 곱연산. 판정 순서가 핵심.

최종 피해 = ⌊ 기본피해 × 스킬배율 × HP감쇠
    × (1 + Σ 주피증)     ← 합연산
    × Π (1 - 받피감)   ← 곱연산
    × 크리티컬 × 블럭 × (1-DR) × 피로도 × 난수(0.95~1.05) ⌋

판정 순서 (7단계)

단계판정설명
①ACC vs EVD회피 성공 시 피해 = 0 (MISS)
②BLK 판정블럭 성공 시 피해 × (1 - BRD)
③DEF/PEN 연산기본 피해 = ATK + max(0, ATK-DEF)×0.5
④버프/디버프주피증(합산), 받피감(곱산)

🚶 주둔 캐릭터 모션 가이드 (아티스트용)

배럭의 생활 모션(요리/연구/식사)과 달리, 거점에서는 군사적 활동 모션이 중심. 기존 캐릭터 스프라이트에 아래 모션을 추가 제작.

🚶
순찰 (Patrol)
• 성벽 위 또는 거점 외곽을 좌→우 왕복
• 걸음: 보통 속도, 어깨에 창 메거나 손에 등불
• 2~3보 걸을 때마다 좌우 둘러보기 (고개 돌림)
• 야간: 한 손에 등불 들고 순찰, 등불 빛 원형 glow
• 프레임: 8프레임 루프 + 둘러보기 4프레임
🧍
경비 (Guard)
• 성문/시설 앞에서 직립 자세
• 양손으로 창 수직 파지, 또는 팔짱
• 3~5초마다 미세 체중 이동 (발 바꿈)
• 간헐적 주변 응시 (좌→정면→우, 느리게)
• 경보 시: 경계 자세 전환 (한 손 검에, 몸 긴장)
• 프레임: 4프레임 idle + 경계 전환 4프레임
⚔️
훈련 (Training)
• 훈련장/주둔지 앞에서 허수아비 타격
• 검 휘두르기 → 돌아오기 → 다시 휘두르기 루프
• 타격 시 먼지 파티클 + 허수아비 흔들림
• 변형: 2인 대련 모션 (서로 마주보고 공격/방어)
• 프레임: 6프레임 루프 + 타격 이펙트
🔭
감시 (Lookout)
• 감시탑/성벽 위에서 먼 곳 응시
• 손 이마에 대고 원거리 응시 자세
• 망원경 사용 변형 (Tier2+ 감시탑)
• 간헐적 뒤돌아 아래로 신호 보내기
• 프레임: 4프레임 idle + 신호 3프레임
💪
작업 (Labor)
• 보급소/성벽에서 상자 나르기 / 벽 보수
• 상자 들기 → 옮기기 → 내려놓기 루프
• 변형: 망치질(성벽 보수), 밧줄 당기기(크레인)
• 땀 파티클 (간헐)
• 프레임: 8프레임 루프
🏃
긴급 집합 (Alert)
• 경보 발동 시 모든 캐릭터 달려감
• 현재 위치 → 성문/방어 위치로 달리기
• 무기 뽑으며 달리기 (검 뽑기 1프레임)
• 도착 후 경비 자세로 전환
• 프레임: 6프레임 달리기 + 무기뽑기 4프레임
